# rutinapp 🟣

Planificador de hábitos y rutinas diarias hecho con HTML, CSS y JavaScript puro. Sin frameworks, sin instalaciones, sin servidor. Abre el archivo y listo.

## ¿Qué incluye?

- Pantalla de bienvenida con nombre de usuario guardado en local
- Vista semanal con el día actual resaltado
- Marcar y desmarcar hábitos con racha automática
- Estadísticas en tiempo real: completados, racha máxima y media
- Barra de progreso diaria
- Mapa de calor semanal por hábito
- Añadir y eliminar hábitos con categorías
- Modo claro y oscuro
- Saludo personalizado según la hora del día

## Cómo usarlo

1. Descarga o clona el repositorio
2. Abre `rutinapp.html` en cualquier navegador
3. No necesita internet ni instalación

## Tecnologías

- HTML5
- CSS3 con variables (sin frameworks)
- JavaScript vanilla
- localStorage para persistencia de datos
